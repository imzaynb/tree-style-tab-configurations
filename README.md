# TreeStyleTab Configurations

I like to use the TreeStyleTab plugin for Firefox to organize my tabs. Here is my minimal configuration to remove the firefox tab bar and make the TST tabs more compact. 

## `userChrome.css`
In order to place this file correctly, you must complete the following steps:

1. Navigate to `Help > More troubleshooting information > Profile Folder > Open Folder`.
2. Within this folder, create a folder called `chrome`. 
3. Within `chrome/`, add the file called `userChrome.css`

## `treestyletab.css`
This is added to the configuration of TreeStyleTab. To do this, follow these steps:

1. Select the puzzle piece in the tool bar.
2. Right click on TreeStyleTab and select `Manage Extension`.
3. Select the `Options` pane on this page.
4. Under `Advanced > User Style Sheet`, in the editable text area, either copy in the contents of `treestyletab.css`, or select `Load from File` at the bottom.

And thats it! Enjoy.
